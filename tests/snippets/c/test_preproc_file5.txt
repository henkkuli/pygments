Preprocessor macros should appear only at the beginning of the line.
Otherwise we should produce an error token.

---input---
foo(); #define FOO 0

---tokens---
'foo'         Name
'('           Punctuation
')'           Punctuation
';'           Punctuation
' '           Text
'#'           Error
'define'      Name
' '           Text
'FOO'         Name
' '           Text
'0'           Literal.Number.Integer
'\n'          Text
